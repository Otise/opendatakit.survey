This Eclipse Android project has dependencies upon these other projects:

actionbarsherlock  https://mitchellsundt@code.google.com/p/opendatakit.actionbarsherlock/ 
playlicensing      https://mitchellsundt@code.google.com/p/opendatakit.playlicensing/ 
androidcommon      https://mitchellsundt@code.google.com/p/opendatakit.androidcommon/ 

as does ODK Tables. If any of the above projects are changed, it is important that 
you also update ODK Tables to support the changes to these libraries.
