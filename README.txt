This Eclipse Android project has dependencies upon these other projects:

opendatakit.playservices      https://mitchellsundt@code.google.com/p/opendatakit.playservices/
opendatakit.androidlibrary      https://mitchellsundt@code.google.com/p/opendatakit.androidlibrary/
opendatakit.androidcommon      https://mitchellsundt@code.google.com/p/opendatakit.androidcommon/

as does ODK Tables and ODK Sync. If any of the above projects are changed, it is important that
you also update ODK Tables and ODK Sync to support the changes to these libraries.
